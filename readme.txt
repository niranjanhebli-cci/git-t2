Steps to Clone a Repository:-

Using Command Line:-

1) Open your GitHub account and view the repository you would like to clone 

2) Under the Code option, Copy the HTTPS link provided.

3) Open new folder where you want to clone your repository

4) Open terminal in your new folder and use the command:-

 	git clone <HTTPS link of the repository>

for the given task, the command was :-

	git clone https://github.com/niranjanhebli-cci/git-t2.git
